App: POKEMON BATTLE SIMULATOR
Version: 1
Autor: James Liliou

***Roles:
A Simple Pokemon game with MessageBox

***Requirements:
Requires nothing.

***Message:
This game has just 5 mons and 10 levels. But it is editable. Notify:
- app/consts.ch contains constants like critical hit rate
- app/res/mons.ch contains the Pokemon of the game
- app/res/levels.ch contains the levels of the game
- app/res folder contains the images of the game
- app/screens folder contains the screens of the game, home.ch is the main one.
- app/scripts folder contains the mecanics of the game in gcs scripts files.

Note : You better save before any edition.
